Para autenticarte en la API de Gates2b, necesitas obtener una **API Key**. Sigue los pasos abajo:

1. Accede al [panel de control de Gates2b](https://painel.gates2b.com).
2. Navega a la pestaña **Integracion**.
3. Genera y copia tu **API Key**.

Usa esta **API Key** para autenticar tus solicitudes a la API de Gates2b. La **API Key** debe incluirse en el header de la solicitud como Authorization `API Key`:

```http
API Key: {tu-api-key}
```

## Uso de la API

Puedes usar la **API Key** para autenticarte en la API de Gates2b. Incluye la **API Key** en el header de tu solicitud como `API Key`.

Recuerda que el uso de la API esta sujeto a los [Terminos de Servicio](https://gates2b.com) y la [Politica de Uso Aceptable](https://gates2b.com).
