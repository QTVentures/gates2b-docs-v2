## Por que liberar IP o URL?

La liberacion de IP o URL es una capa esencial de seguridad que asegura que solo servidores autorizados puedan acceder a nuestras APIs. Este control protege tus datos y evita accesos no autorizados.

## Quien necesita liberar?

- **Cuentas brasileñas:** La liberacion de IP o URL es **obligatoria**.
- **Cuentas uruguayas:** La liberacion de IP o URL es **opcional**, pero altamente recomendada para mayor seguridad.

## Como liberar IP o URL

La liberacion de IP o URL es simple y rapida. Sigue los pasos abajo:

1. **Accede al Panel de Integracion:** Entra en [painel.gates2b.com](https://painel.gates2b.com).
2. **Navega a Integracion:** En la barra de navegacion, haz clic en **Integracion**.
3. **Agrega el IP o URL:** Selecciona la opcion para agregar un nuevo IP o URL y completa la informacion solicitada.
4. **Guarda los cambios:** Confirma el registro y listo. Tu IP o URL quedara liberado para usar las APIs.

<Tip>
  Si necesitas liberar multiples IPs o URLs, solo repite el proceso. Si tienes
  dudas, estamos listos para ayudar.
</Tip>
