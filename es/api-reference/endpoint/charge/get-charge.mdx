## Otras consultas disponibles

### Listar Cobros

`GET /charge/list?page=1&limit=10`

Retorna la lista paginada de cobros del usuario autenticado.

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `page` | number | Número de página (por defecto: 1) |
| `limit` | number | Registros por página (por defecto: 10) |

### Buscar por Referencia Externa

`GET /charge?externalReference=pedido-123`

Busca un cobro por la referencia externa de su sistema.

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `externalReference` | string | Referencia externa usada en la creación |

### Listar Intentos de un Cobro

`GET /charge/{chargeId}/attempts`

Retorna todos los intentos de pago asociados a un cobro. Cada intento contiene:

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `id` | string | ID del intento |
| `chargeId` | string | ID del cobro |
| `paymentMethod` | string | Método usado (`PIX`, `CREDIT_CARD`, `INTERNATIONAL`) |
| `status` | string | Estado del intento |
| `checkoutMeta` | object | Metadatos del gateway |

### Buscar Intento Específico

`GET /charge/attempts/{attemptId}`

Retorna los detalles de un intento específico, incluyendo estado del gateway y `checkoutMeta`.
