<ResponseField name="id" type="string">

<ResponseField name="currency" type="string">

## Crear Intento Independiente

`POST /charge/{chargeId}/attempts`

Crea un nuevo intento de pago para un cobro existente (server-side, autenticado vía JWT).

| Parámetro | Tipo | Obligatorio | Descripción |
|-----------|------|:-----------:|-------------|
| `paymentMethod` | string | Sí | Método de pago (`PIX`, `INTERNATIONAL`) |
| `currency` | string | Condicional | Moneda de destino (obligatorio para `INTERNATIONAL`) |
| `attemptIdempotencyKey` | string | Sí | Clave de idempotencia |

```bash
curl -X POST "https://api.gates2b.com/charge/{chargeId}/attempts" \
  -H "Authorization: Bearer {su-token}" \
  -H "Content-Type: application/json" \
  -d '{
    "paymentMethod": "PIX",
    "attemptIdempotencyKey": "uuid-v4"
  }'
```
