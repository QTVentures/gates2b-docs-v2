---
title: "Crear Pago"
sidebarTitle: "/payments"
api: "POST https://api.gates2b.com/payments"
authMethod: "authorization"
description: "Este endpoint crea un pago con vencimiento configurable."
---

<ParamField
  header="Authorization"
  type="string"
  placeholder="Bearer {token}"
  required
>
  El token de autenticacion en formato Bearer es obligatorio para acceder a este
  endpoint.
</ParamField>

<ParamField
  body="clientMode"
  type="string"
  placeholder="noNeedToSpecify | fillDataNow | requestCheckout"
  required
>
  Define el modo de tratamiento de los datos del cliente. - `noNeedToSpecify`:
  No requiere datos del cliente. - `fillDataNow`: Los datos del cliente deben
  enviarse ahora. - `requestCheckout`: Los datos del cliente se completaran via
  checkout.
</ParamField>

<ParamField body="amount" type="number" placeholder="150.50" required>
  Valor del pago.
</ParamField>

<ParamField
  body="expire_at"
  type="string"
  placeholder="2025-12-31T23:59:59Z"
  required
>
  Fecha y hora de expiracion en formato ISO 8601.
</ParamField>

<ParamField
  body="description"
  type="string"
  placeholder="Descrição do pagamento"
>
  Descripcion del pago. Campo opcional.
</ParamField>

<ParamField
  body="external_reference"
  type="string"
  placeholder="Referência externa"
>
  Referencia externa de la aplicacion. Campo opcional.
</ParamField>

<ParamField
  body="redirect_url"
  type="string"
  placeholder="https://sua-url.com/retorno"
>
  URL de redireccion despues del pago. Campo opcional.
</ParamField>

<ParamField body="phoneNumber" type="string" placeholder="+5511999999999">
  Numero de telefono del cliente. Campo opcional.
</ParamField>

<ParamField body="email" type="string" placeholder="cliente@email.com">
  Correo del cliente. Campo opcional.
</ParamField>

<ParamField body="document_type" type="string" placeholder="CPF | CNPJ">
  Tipo de documento del cliente. Campo opcional.
</ParamField>

<ParamField body="name_client" type="string" placeholder="Nome do cliente">
  Nombre del cliente. Campo opcional.
</ParamField>

<ParamField body="document_client" type="string" placeholder="12345678900">
  Documento del cliente. Campo opcional.
</ParamField>
