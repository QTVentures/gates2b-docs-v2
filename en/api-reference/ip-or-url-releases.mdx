## Why allowlist an IP or URL?

IP or URL allowlisting is an essential security layer that ensures only authorized servers can access our APIs. This control protects your data and prevents unauthorized access.

## Who needs to allowlist?

- **Brazilian Accounts:** IP or URL allowlisting is **mandatory**.
- **Uruguayan Accounts:** IP or URL allowlisting is **optional**, but highly recommended for better security.

## How to allowlist an IP or URL

Allowlisting an IP or URL is quick and simple. Follow the steps below:

1. **Access the Integration Panel:** Go to [painel.gates2b.com](https://painel.gates2b.com).
2. **Navigate to Integration:** In the navigation bar, click **Integration**.
3. **Add the IP or URL:** Select the option to add a new IP or URL and fill in the requested information.
4. **Save the changes:** Confirm the entry and you are done! Your IP or URL will be allowlisted for API usage.

<Tip>
  If you need to allowlist multiple IPs or URLs, just repeat the process. If you
  have any questions, we are ready to help.
</Tip>
