<ResponseField name="externalReference" type="string">

## Create Standalone Attempt

`POST /charge/{chargeId}/attempts`

Creates a new payment attempt for an existing charge (server-side, authenticated via JWT).

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `paymentMethod` | string | Yes | Payment method (`PIX`, `INTERNATIONAL`) |
| `currency` | string | Conditional | Target currency (required for `INTERNATIONAL`) |
| `attemptIdempotencyKey` | string | Yes | Idempotency key |

```bash
curl -X POST "https://api.gates2b.com/charge/{chargeId}/attempts" \
  -H "Authorization: Bearer {your-token}" \
  -H "Content-Type: application/json" \
  -d '{
    "paymentMethod": "PIX",
    "attemptIdempotencyKey": "uuid-v4"
  }'
```
