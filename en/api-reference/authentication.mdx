To authenticate to the Gates2b API, you need to obtain an **API Key**. Follow the steps below:

1. Access the [Gates2b control panel](https://painel.gates2b.com).
2. Navigate to the **Integration** tab.
3. Generate and copy your **API Key**.

Use this **API Key** to authenticate your requests to the Gates2b API. The **API Key** must be included in the request header as Authorization `API Key`:

```http
API Key: {your-api-key}
```

## API Usage

You can use the **API Key** to authenticate to the Gates2b API. Include the **API Key** in your request header as `API Key`.

Remember that API usage is subject to the [Terms of Service](https://gates2b.com) and the [Acceptable Use Policy](https://gates2b.com).
