## Criar Tentativa Avulsa

`POST /charge/{chargeId}/attempts`

Cria uma nova tentativa de pagamento para uma cobrança existente (server-side, autenticado via JWT).

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `paymentMethod` | string | Sim | Método de pagamento (`PIX`, `INTERNATIONAL`) |
| `currency` | string | Condicional | Moeda de destino (obrigatório para `INTERNATIONAL`) |
| `attemptIdempotencyKey` | string | Sim | Chave de idempotência |

```bash
curl -X POST "https://api.gates2b.com/charge/{chargeId}/attempts" \
  -H "Authorization: Bearer {seu-token}" \
  -H "Content-Type: application/json" \
  -d '{
    "paymentMethod": "PIX",
    "attemptIdempotencyKey": "uuid-v4"
  }'
```
