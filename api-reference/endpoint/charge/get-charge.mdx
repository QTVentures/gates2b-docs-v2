## Outras consultas disponíveis

### Listar Cobranças

`GET /charge/list?page=1&limit=10`

Retorna a lista paginada de cobranças do usuário autenticado.

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `page` | number | Número da página (padrão: 1) |
| `limit` | number | Registros por página (padrão: 10) |

### Buscar por Referência Externa

`GET /charge?externalReference=pedido-123`

Busca uma cobrança pela referência externa do seu sistema.

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `externalReference` | string | Referência externa usada na criação |

### Listar Tentativas de uma Cobrança

`GET /charge/{chargeId}/attempts`

Retorna todas as tentativas de pagamento associadas a uma cobrança. Cada tentativa contém:

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | string | ID da tentativa |
| `chargeId` | string | ID da cobrança |
| `paymentMethod` | string | Método usado (`PIX`, `CREDIT_CARD`, `INTERNATIONAL`) |
| `status` | string | Status da tentativa |
| `checkoutMeta` | object | Metadados do gateway |

### Buscar Tentativa Específica

`GET /charge/attempts/{attemptId}`

Retorna os detalhes de uma tentativa específica, incluindo status do gateway e `checkoutMeta`.
