## Servidor MCP hospedado

Acesse o servidor MCP. Use a URL abaixo para conectar aplicações de IA ao conteúdo desta documentação.

<Info>

</Info>

Conecte clientes de IA (por exemplo, Cursor, Claude ou outras aplicações compatíveis com MCP) apontando para essa URL, conforme a configuração do seu produto.
